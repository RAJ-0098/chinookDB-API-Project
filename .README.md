# Chinook Analytics REST API

A backend REST API project built using FastAPI and PostgreSQL on top of the Chinook sample database.

This project was created to understand real backend engineering concepts including:
- REST APIs
- PostgreSQL integration
- SQL query execution
- API testing
- pytest
- fixtures
- markers
- hooks
- mock testing
- TestClient
- integration testing
- containerized testing concepts

---

# Project Architecture

Client / Browser
        ↓
FastAPI Backend
        ↓
SQLAlchemy
        ↓
PostgreSQL (Chinook Database)

---

# Tech Stack

## FastAPI
- Python backend framework
- Used for building REST APIs
- Handles routing, requests, responses, JSON conversion

## PostgreSQL
- Relational database system
- Stores Chinook database data
- Used for SQL queries and relationships

## SQLAlchemy
- Database interaction layer
- Connects FastAPI with PostgreSQL
- Executes SQL queries safely

## pytest
- Python testing framework
- Used for API testing and backend testing

## TestClient
- Mock API request client
- Simulates HTTP requests internally
- Used for API testing without browser/Postman

## unittest.mock
- Mocking framework
- Used to replace real dependencies with fake objects during testing

# Run Project

Install dependencies:

pip install -r requirements.txt

Run FastAPI server:

uvicorn app.main:app --reload

Open Swagger Docs:

http://127.0.0.1:8000/docs

---

# Run Tests

pytest

Run smoke tests only:

pytest -m smoke
